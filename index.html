Sweet Eclipse

Eclipse is an amazing open source platform and a powerful IDE for dozens of programming languages whose user interface hasn’t changed a lot for over 10 years. Does this mean that the Eclipse user experience is stuck in the beginning of the century?

Sweet Eclipse is a project in which the reliable Eclipse platform meets modern UI techniques to provide better user experience. Is this even possible? Check out Sweet Java IDE and you'll be quite surprised.
