Sweet Eclipse

Eclipse is an amazing open source platform and powerful IDE for dozens of programming languages whose user interface doesn't change a lot for more than 10 years. Does this mean that Eclipse user experience stuck at the beginning of the century?

Sweet Eclipse is a project where reliable Eclipse platform meets modern UI techniques to provide better user experience. Is it even possible? Check out Sweet Java IDE and you'll be quite surprised.
